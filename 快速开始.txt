🎯 立即开始游戏：
─────────────────────────────────────────────
方式1：直接打开
  👉 双击打开 singleplayer.html

方式2：从多人模式切换
  👉 打开 index.html
  👉 点击右上角"🎮 单人模式"按钮

🎮 游戏玩法：
─────────────────────────────────────────────
1. 选择难度（😊简单 / 🤔中等 / 🔥困难）
2. 查看题目和频谱对（例如：寒冷 ↔ 炎热）
3. 拖动滑条或点击弧线猜测答案位置
4. 点击"提交答案"
5. 查看得分和正确答案
6. 点击 👍 或 👎 评价题目
7. 点击"下一题"继续游戏

🎯 计分规则：
─────────────────────────────────────────────
💯 完美命中    → 100分 + 礼炮特效
✅ 合理范围    → 60分
😊 接近（≤20） → 30分
😢 未命中      → 10分

📊 题库内容：
─────────────────────────────────────────────
• 简单题10道：极端位置，人人皆知
  例：南极洲（寒冷↔炎热）

• 中等题10道：主观判断，需要思考
  例：冰箱（无用↔必需）

• 困难题10道：哲学辩论，充满争议
  例：自由意志存在（客观真实↔客观虚假）

📝 创作新题目：
─────────────────────────────────────────────
1. 打开 creator_template.md 阅读完整指南
2. 编辑 question_bank.json 添加新题目
3. 使用以下格式：

{
  "id": "easy_011",
  "topic_pair": "左端 ↔ 右端",
  "target_position": 50,
  "difficulty": "easy",
  "range_hint": [30, 70],
  "question_text": "你的题目",
  "question_creator": "你的名字",
  "explanation_for_creator": "设计思路"
}
